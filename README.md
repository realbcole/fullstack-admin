## üßê About <a name = "about"></a>
 MERN Stack Admin Dashboard. Frontend uses React, Material UI, Nivo Charts, Redux Toolkit and Redux Toolkit Query. Backend uses Node JS, Express Js, Mongoose, and MongoDB.
